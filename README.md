# Employee Salary and Retention Analysis

###  Author: Nitesh Kumar

This project performs a full statistical analysis on simulated employee data.  
It helps understand salary patterns, education impact, attrition behavior, and 
performance in organizations using data science techniques.

---

##  Key Analysis Performed

 Salary distribution & descriptive statistics  
 Attrition rate by education (stacked bar chart)  
 Correlation analysis of Salary, Experience & Age  
 Central Limit Theorem demonstration  
 Hypothesis testing:
 One-sample t-test
 Two-sample t-test
 Two-proportion z-test  

 Predictive Modeling:
 Multiple Linear Regression (Salary prediction)
 Poisson Regression (Project count estimation)
